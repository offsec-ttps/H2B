# Debt Management

**bolt.dev prompt**

```
Build a separate page for: "Debt Management"

Features:

Debt Name / Source

Amount Borrowed

Amount Paid

Due Date

Interest Rate (%) or mark as Interest-Free

Priority Level (High, Medium, Low â€“ for repayment order)

Total Debt

Amount Paid

Remaining Due

Number of Active Debts

Highest Priority Debt

Progress Visualization:

Bargraph or Meter per debt showing repayment progress:

Green if >50% paid

Red if <50% paid

Debt List View with sorting/filtering options:

Sort by: Priority, Due Date, Interest Rate, Amount Due

Filter: Interest-Free, High Interest, Paid vs Unpaid

Ask for Help Section:

Add a list of people you trust to reach out to

Include contact, last approached date, and notes

Visibility toggle for privacy

Recommendations Panel:

Tips to reduce debt faster

Reminders to negotiate interest or refinance

Suggest saving goals or budgeting advice

Alert if high-interest debt remains unpaid beyond 50%

Optional Features:

Reminder alerts for due payments

Export debt report (PDF/CSV)

Sync with calendar for repayment dates
```
